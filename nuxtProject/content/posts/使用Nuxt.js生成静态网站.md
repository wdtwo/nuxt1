## 1. 全局安装Nuxt.js
```bash
npm install -g nuxt
```
## 2. 新建项目
```bash
nuxt init demo1
```
## 3. 运行项目
```bash
cd my-static-site
npm run dev
```
## 添加页面
在根目录创建`pages`文件夹,nuxt会自动配置路由
