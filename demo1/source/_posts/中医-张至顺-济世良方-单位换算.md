剂量换算

<!--more-->
------


（由于古代度量衡计量方式的变迁，各个时期的中药剂量换算方法有所不同。）

### 汉代

- 1斤 = 16两 = 248克（约等于250克）
- 1两 = 24铢 = 15.5克（约等于15克）
- 1铢 = 0.625克

一方寸匕：金石类药约2克，草木类药约1克。
- 1升 = 10合 = 200毫升
- 1合 = 20毫升

### 唐宋

唐宋医药按古制，换算方法同汉代。

### 明清

- 1斤 = 16两 = 600克
- 1两 = 37.5克

### 近代

1. 港台：换算方式同清明。
2. 大陆
    - 1斤 = 16两 = 500克
    - 1两 = 10钱 = 30克
    - 1钱 = 10分 = 3克
    - 1分 = 0.3克

### 现代

- 1公斤 = 2斤 = 1000克
- 1斤 = 10两 = 500克
- 1两 = 10钱 = 50克
